rpm、yum、apt-get

发表于 2018-02-08 | 分类于 Linux

rpm包和deb包是两种Linux系统下最常见的安装包格式。
rpm包主要应用在RedHat系列包括 Fedora等发行版的Linux系统上，
deb包主要应用于Debian系列包括现在比较流行的Ubuntu等发行版上。

rpm/yum适用于Redhat、CentOS、Suse等平台；
apt-get/dpkg适用于Debian、Ubuntu等平台；
zypper适合于Suse平台。

rpm常用命令

rpm -e 软件名称
rpm -qa | grep 软件名称
rpm -qa 查看所有已经安装的包
rpm -qi pkg_name 查看已安装包的详细信息，例如安装时间
rpm -qf file_name 查看一个文件属于哪个rpm 包
rpm -ql pkg_name 查看已安装的包里面有哪些文件
rpm -qpl pkg_name 查看一个RPM包里有哪些文件
rpm -Uvh pkg_name 安装所有指定的包，会覆盖原包
rpm -ivh pkg_name 安装所有指定的包，通常会保留原包
rpm -Fvh pkg_name 只安装比当前版本更高的包

yum常用命令(Redhat安装rpm包)

yum install 全部安装
yum install 包名安装指定的安装包
yum groupinsall group1 安装程序组group1
yum update全部更新
yum update包名更新指定程序包
yum check-update检查可更新的程序
yum upgrade包名升级指定程序包
yum groupupdate group1升级程序组
yum info包名显示按装包信息
yum list显示所有已安装和可以安装的程序包
yum list包名显示指定程序包的安装情况
yum groupinfo group1显示程序组group1信息yum search string根据安装包string查找安装包。
yum groupremove group1删除程序组group1
yum deplist package1查看程序package1的依赖情况
yum clean packages清除缓存目录下的软件包
yum clean headers清除缓存目录下的heades
yum clean oldheaders清除缓存目录下的旧的标题
yum clean，yum clean all（= yum clean package; yum clean oldheaders）清除缓存目录下的软件包及旧的heades
yum makecache 更新新的包

apg-get常用命令（Ubantu安装deb包）

apt-cache search package 搜索软件包
apt-cache show package 获取包的相关信息，如说明、大小、版本等
sudo apt-get install package 安装包
sudo apt-get install package –reinstall 重新安装包
sudo apt-get -f install 修复安装
sudo apt-get remove package 删除包
sudo apt-get remove package –purge 删除包，包括配置文件等
sudo apt-get update 更新源
sudo apt-get upgrade 更新已安装的包
sudo apt-get dist-upgrade 升级系统
apt-cache depends package 了解使用该包依赖那些包
apt-cache rdepends package 查看该包被哪些包依赖
sudo apt-get build-dep package 安装相关的编译环境
apt-get source package 下载该包的源代码
sudo apt-get clean && sudo apt-get autoclean 清理无用的包
sudo apt-get check 检查是否有损坏的依赖
